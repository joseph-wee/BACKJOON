# 문제 2438



# 풀이

```javascript
let fs = require("fs");
let input = fs.readFileSync("dev/stdin").toString().split("\n")

let N = parseInt(input[0])    // 출력할 별의 Line 수
let star = "";    // 한 줄에 출력될 별의 개수

for (i = 0; i < N; i++) {
  star += "*";
  console.log(star)
}
```
